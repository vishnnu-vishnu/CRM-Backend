# 📊 CRM Daily Activities Platform

A simple CRM (Customer Relationship Management) application for tracking daily activities, tasks, and workflow progress.  
Similar to Jira, but designed for lightweight daily task management.

---

## 🚀 Features

- Create, update, and delete daily tasks
- Assign tasks to users/team members
- Track task status (To Do, In Progress, Done)
- Daily activity logging
- Task comments and updates
- Search and filter tasks
- Basic dashboard overview

---

## 🏗️ Tech Stack

- Frontend: React / Angular / Vue (choose your stack)
- Backend: Node.js / Express / Django (choose your stack)
- Database: MongoDB / MySQL / PostgreSQL
- Authentication: JWT / Session-based auth
